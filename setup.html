⚠️ Fix your Firestore Security Rules first

Go to Firebase Console → Firestore → Rules and paste these rules, then click Publish:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    // Allow authenticated users to read/write their own user doc
    match /users/{userId} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }

    // Allow authenticated users to read all other collections
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
⚠️ Delete this file from your server after setup
